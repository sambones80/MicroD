auto: Size based on content fr: Size based on fraction of available space in container repeat(number, unit): sets a number of tracks, sets a unit of size minmax(#px, #px): Sets minimum/maximum size, fr can be used as max auto-fill: Used in repeat() to set the number of tracks that will fit in container. Set a min px size to make responsive by wrapping to next row. Will add empty tracks if not enough elements to fill container. auto-fit: Same as auto-fill but empty tracks have no width, use with minmax() Set a max size 1fr to make responsive by expanding all tracks to fit container.
